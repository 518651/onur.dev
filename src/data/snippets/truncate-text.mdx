ogTitle: '**Truncate Text**'
---

Let's say you want to truncate the text within 4 lines.

```css
.truncate-4 {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 4;
  overflow: hidden;
}
```
