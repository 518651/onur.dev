---
title: 'useHasMounted()'
summary: 'Useful for telling if the app has mounted'
publishedAt: '2021-01-24'
ogTitle: '**useHasMounted**'
---

```js
function useHasMounted() {
  const [hasMounted, setHasMounted] = React.useState(false)

  React.useEffect(() => {
    setHasMounted(true)
  }, [])

  return hasMounted
}

// Usage
const SomeClientSideOnlyComponent = () => {
  const hasMounted = useHasMounted()
  if (!hasMounted) return null
  return <>Client only!</>
}
```
